# supermegaCAD

Графический редактор для работы с геометрическими фигурами.

## Запуск

```bash
python main.py
```

## Основные возможности

- **Рисование отрезков** - мышью или через координаты
- **Выделение и удаление** фигур
- **Навигация** - панорамирование, масштабирование, поворот вида
- **Декартовы и полярные координаты**
- **Настройка сетки и единиц угла**

## Управление

### Инструменты:
- `D` - Рисование
- `S` - Выделение  
- `H` - Панорамирование (Рука)

### Навигация:
- `+` / `=` - Увеличить масштаб
- `-` - Уменьшить масштаб
- `F` - Показать все фигуры
- `[` - Поворот на 90° вправо
- `]` - Поворот на 90° влево
- `Home` - Сброс вида
- **Колесико мыши** - Масштабирование
- **СКМ** - Панорамирование
- **ПКМ** - Контекстное меню

### Редактирование:
- `Delete` / `Backspace` - Удалить фигуру

## Интерфейс

- **Панель инструментов** - выбор инструментов, масштаб, поворот (вкладки: Главная, Рисование, Вид, Инструменты)
- **Правая панель** - ввод координат, кнопки действий
- **Строка состояния** - координаты курсора, масштаб, угол поворота, активный инструмент
- **Контекстное меню (ПКМ)** - быстрый доступ к командам

## Описание файлов

### Основные файлы

**`main.py`** - Главный файл приложения
- Инициализация UI (окно, меню, панели инструментов)
- Управление инструментами и событиями
- Обработка горячих клавиш и мыши
- Координация работы всех компонентов

**`view_transform.py`** - Система аффинных преобразований
- Класс `ViewTransform` для управления видовой матрицей
- Преобразование координат (world_to_screen, screen_to_world)
- Панорамирование, масштабирование, поворот вида
- Подгонка вида под все фигуры

**`core.py`** - Основные компоненты
- `CoordinateConverter` - конвертация координат (полярные ↔ декартовы)
- `SegmentGeometry` - геометрические расчёты для отрезков
- `CanvasRenderer` - отрисовка сетки, осей и фигур на canvas
- `UIBuilder` - вспомогательные функции для создания UI элементов

### Модуль shapes/ (Геометрические фигуры)

**`shapes/base.py`** - Базовый класс для всех фигур
- Абстрактный класс `Shape` с обязательными методами
- `draw()` - отрисовка фигуры
- `distance_to_point()` - расстояние до точки
- `get_info()` - информация о фигуре
- `get_bounds()` - границы фигуры
- `translate()` - перемещение фигуры
- `to_dict()` / `from_dict()` - сериализация

**`shapes/segment.py`** - Класс отрезка
- Реализация класса `Segment` (отрезок между двумя точками)
- Отрисовка с учётом трансформаций вида
- Расчёт расстояния от точки до отрезка
- Информация об отрезке (координаты, длина, угол)

**`shapes/__init__.py`** - Экспорт классов фигур
- Импорт и экспорт `Shape` и `Segment`

### Модуль tools/ (Инструменты)

**`tools/base.py`** - Базовый класс для всех инструментов
- Абстрактный класс `Tool` с методами обработки событий
- `on_mouse_down()` - нажатие кнопки мыши
- `on_mouse_move()` - движение мыши
- `on_mouse_up()` - отпускание кнопки мыши
- `on_activate()` / `on_deactivate()` - активация/деактивация
- `get_cursor()` - курсор для инструмента

**`tools/draw_tool.py`** - Инструмент рисования
- Рисование отрезков левой кнопкой мыши
- Предпросмотр отрезка при рисовании
- Создание отрезков из двух точек

**`tools/select_tool.py`** - Инструмент выделения
- Выделение фигур левой кнопкой мыши
- Поиск ближайшей фигуры к точке клика
- Визуальное выделение (зелёный цвет, увеличенная толщина)

**`tools/pan_tool.py`** - Инструмент панорамирования
- Панорамирование холста при зажатии ЛКМ
- Инструмент "Рука" для перемещения вида
- Изменение курсора во время панорамирования

**`tools/navigation_handler.py`** - Обработчик навигации
- Обработка средней кнопки мыши (СКМ) для панорамирования
- Обработка колесика мыши для масштабирования
- Масштабирование относительно курсора

**`tools/__init__.py`** - Экспорт классов инструментов
- Импорт и экспорт всех инструментов

### Модуль managers/ (Менеджеры)

**`managers/shape_manager.py`** - Менеджер фигур
- Управление коллекцией всех фигур на холсте
- Добавление и удаление фигур
- Выделение и снятие выделения
- Экспорт/импорт фигур (сериализация)
- Получение информации о фигурах

**`managers/__init__.py`** - Экспорт классов менеджеров
- Импорт и экспорт `ShapeManager`
